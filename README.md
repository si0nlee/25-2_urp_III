**팀명: SOMA**

> **2025-2학기 URP III형**

## 📌 Project Overview
본 연구는 인간-컴퓨터 상호작용(HCI) 관점에서 **사람의 외향성이 상대방에 대한 호감이나 인상을 형성하는 '타이밍'에 어떠한 영향을 미치는지**를 분석합니다. 행동 신호와 성격 데이터를 결합하여 대인 매력도 형성 과정을 데이터 과학적으로 규명하는 것을 목표로 합니다.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** PyTorch, Pandas, Scikit-learn, Matplotlib
- **Dataset:** MatchNMingle Dataset (Speed Dating Behavioral Data)
- **Domain:** Human-Computer Interaction (HCI), Computer Vision
